Building a Tool on Top of OpenMDAO
----------------------------------

There are several things to take into account when setting up a repository that depends on
OpenMDAO.  Repository structure, writing project documentation with Sphinx, getting project documentation transferred
to github-pages, setting up continuous integration on Travis CI, and releasing software are the topics covered.
